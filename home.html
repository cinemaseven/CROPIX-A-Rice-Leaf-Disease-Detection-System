About this Website

This website serves as the official digital platform and deployment interface for a software project that provides an accessible, high-precision tool for rapid crop health assessment.

How to Use

  1. Prepare a rice leaf image.
  2. Select or update the date using the calendar icon above to match when the image was taken.
  3. Drag and drop the image or click Choose file to upload.
  4. Upload one image at a time.
  5. The disease protection and confidence score will be displayed automatically.
  6. Click the info icon for more information about the results.
  7. To analyze another image, simply repeat the steps aboved
